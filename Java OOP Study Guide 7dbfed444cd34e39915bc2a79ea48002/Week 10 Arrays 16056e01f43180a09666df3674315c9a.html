Arrays

1. Primitive vs. Reference Type

Primitive Type:

Stores a single value (e.g., int, double).

int num = 5; // Can only hold one integer value

Reference Type:

Stores the address of an object in memory.

Scanner input = new Scanner(System.in); // Reference to a Scanner object

2. Creating, Initializing, and Modifying Arrays

  • Array Basics: Arrays store multiple values of the same type.
    int[] c = new int[12]; // Creates an array of size 12
  • Array Initializer: 
    int[] numbers = {1, 2, 3, 4};
  • Using Arrays: Access elements with indices (c[0], c[1]), starting from 0.
    c[0] = 10; // Sets the first element to 10

3. Exception Handling (Try-Catch Statement)

  • Definition: Handles errors (exceptions) during program execution.
  • Syntax:
    try {
		int[] numbers = {1, 2, 3};
		System.out.println(numbers[10]); // Error
	} catch (ArrayIndexOutOfBoundsException e) {
		System.out.println("Something went wrong.");
	} finally {
		System.out.println("Execution finished.");
	}
	// Output:
	//	Something went wrong.
	//	Execution finished.

4. Enhanced for Statement

  • Definition: Iterates through array elements without using an index.
    int[] numbers = {1, 2, 3};
	for (int num : numbers) {
		System.out.println(num);
	}

5. Passing Arrays to Methods

  • Pass arrays as arguments to methods by specifying their name without any brackets.
    public static void printArray(int[] arr) {
		for (int num : arr) System.out.println(num);
	}
	
	public static void main(String[] args) {
		int[] numbers = {1, 2, 3};
		printArray(numbers);
	}

6. Pass-By-Value vs. Pass-By-Reference

  • Pass-By-Value: A copy of the value is passed to the method. Changes don’t affect the original value.
    public static void modify(int x) {
		x = 10; // Does not affect the original variable
	}
  • Pass-By-Reference:
    • Direct access to the variable’s value, allowing modification.
    • Java uses pass-by-value for primitive types and reference types.

7. Multidimensional Arrays

Definition: Arrays with rows and columns (2D arrays).

int[][] matrix = {{1, 2}, {3, 4}};
	System.out.println(matrix[0][0]); // Output: 1

Flexible Rows:

int[][] matrix = {{1, 2}, {3, 4, 5}};
