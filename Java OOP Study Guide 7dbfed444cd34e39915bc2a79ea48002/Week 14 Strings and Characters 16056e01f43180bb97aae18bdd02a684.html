Strings and Characters

1. Fundamentals of Strings and Characters

  • Characters:
    • Building blocks of Java programs.
    • Represented as a single character in single quotes (e.g., 'a').
    • Characters are stored as integer values using the Unicode character set.
  • Strings:
    • Represented by the String class in Java.
    • A sequence of characters enclosed in double quotes (e.g., "Hello").

2. Class String and Class Character

  • Key Features of the String Class:
    • Immutable: Once created, a string cannot be modified.
    • Constructors for initializing string objects in different ways.
  • String Methods:
    • length(): Returns the length of a string.
    • charAt(index): Returns the character at the specified index.
    • compareTo(String anotherString): Compares strings lexicographically.

    Example:

    String str1 = "Hello";
	String str2 = "Hello";
	System.out.println(str1.compareTo(str2)); // Output: 0
  • Other Useful Methods:
    • equals(), equalsIgnoreCase(), startsWith(), endsWith(), valueOf().
    • Extracting substrings, locating characters, and concatenating strings.
  • Character Class:
    • Provides utilities for working with characters.
    • Methods include isLetter(), isDigit(), toUpperCase(), etc.

3. Tokenizing Strings

  • Definition:
    • Breaking a string into smaller substrings using delimiters.
    • Java provides the StringTokenizer class in java.util.
  • Example:
    import java.util.StringTokenizer;
	StringTokenizer st = new StringTokenizer("Java is fun", " ");
	while (st.hasMoreTokens()) {
		System.out.println(st.nextToken());
	}

4. Regular Expressions

  • Definition:
    • A sequence of characters and symbols defining a pattern for matching strings.
    • Classes used:
      • Pattern: Represents the regular expression.
      • Matcher: Used to perform matching operations.
  • Example:
    import java.util.regex.*;
	public class RegexExample {
		public static void main(String[] args) {
			Pattern p = Pattern.compile(".s"); // . matches any single character
			Matcher m = p.matcher("as");
			System.out.println(m.matches()); // Output: true
		}
	}
  • Regex Quantifiers:
    • {n}: Matches exactly n occurrences.
    • {n,}: Matches n or more occurrences.
    • {n,m}: Matches between n and m occurrences.

    Example:

    System.out.println(Pattern.matches("[a-zA-Z0-9]{6}", "arun32")); // Output: true
