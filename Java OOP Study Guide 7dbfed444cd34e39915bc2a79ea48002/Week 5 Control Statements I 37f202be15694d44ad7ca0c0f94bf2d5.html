Control Statements

1. Algorithms and Pseudocode

  • Algorithm: A step-by-step procedure for solving a problem.
    • Example: Add Two Numbers:
      1. Start
      1. Declare variables num1, num2, and sum.
      1. Read values for num1 and num2.
      1. sum = num1 + num2
      1. Display sum.
      1. Stop
  • Pseudocode: Informal, human-readable description of logic.
    • Does not execute on computers.
If grade >= 60
	   Print "Passed"
	Else
	   Print "Failed"

2. Control Structures

  • Types of Control Structures:
    1. Sequence: Executes statements sequentially.
    1. Selection: Decision making (if, if...else, switch).
    1. Repetition: Loops (while, do...while, for).

3. If Statement

Executes a block of code only if the condition is true.

int age = 20;
	if (age > 18) {
	   System.out.println("Age is greater than 18");
	}

4. If...Else Statement

  • Executes one block when the condition is true; otherwise, another block.
int number = 13;
	if (number % 2 == 0) {
	   System.out.println("Even number");
	} 
	else {
	   System.out.println("Odd number");
	}

5. Nested If Statement

  • Definition: if...else statements within other if...else blocks for multiple conditions.
if (grade >= 90) System.out.println("A");
	else if (grade >= 80) System.out.println("B");
	else if (grade >= 70) System.out.println("C");
	else if (grade >= 60) System.out.println("D");
	else System.out.println("F");

6. Switch...Case Statement

  • Allows multiple values to control program flow.
  • Rules:
    • switch expression must be byte, short, int, long, enum, or String.
    • break is used to exit after matching a case.
    • default runs if no case matches.
int number = 20;
	switch (number) {
	   case 10: System.out.println("10"); break;
	   case 20: System.out.println("20"); break;
	   default: System.out.println("Not in 10 or 20");
	}

7. Conditional Operator (?:)

  • A short form of if...else.
  • Syntax: condition ? trueValue : falseValue;
int a = 6, b = 12;
	String result = (a == b) ? "Yes" : "No";
	System.out.println("Ans: " + result);

8. For and While Loops

  • Definition: Repetition statements allow code execution repeatedly while a condition is true.
  • Components of a Loop:
    1. Control Variable: Variable used for iteration.
    1. Initial Value: Start value of the control variable.
    1. Increment/Decrement: Change in control variable per iteration.
    1. Condition: Determines when to stop looping.
  • Types of Loops:
    • while Loop: Executes while a condition is true.
      int counter = 1;
	while (counter <= 10) {
		System.out.println(counter);
		counter++;
	}

    • for Loop: Combines initialization, condition, and increment in a single line.
      for (int i = 1; i <= 10; i++) {
		System.out.println(i);
	}
	/*
	i =i+1 
	i +=1
	++i
	i++
	*/

    • do...while Loop: Executes the body at least once before checking the condition.
      int counter = 1;
	do {
		System.out.println(counter);
		counter++;
	} while (counter <= 10);

9. Nested Loops

  • Definition: A loop inside another loop.
for (int i = 1; i <= 5; i++) {
		for (int j = 1; j <= 3; j++) {
			System.out.println("i = " + i + ", j = " + j);
		}
	}
  • Use Case: Printing patterns or multi-dimensional operations.

10. Break and Continue Statements

  • break: Immediately exits the loop.
for (int i = 1; i <= 10; i++) {
		if (i == 5) break; // Loop exits when i equals 5
		System.out.println(i);
	}

  • continue: Skips the remaining statements in the loop body for the current iteration and proceeds to the next iteration.
for (int i = 1; i <= 10; i++) {
		if (i == 5) continue; // Skip when i equals 5
		System.out.println(i);
	}

11. Logical Operators

  • Combine conditions to form complex expressions.
OperatorDescriptionExample
&&Conditional AND (both must be true)if (a > 0 && b > 0)
||Conditional OR (only one must be true)if (a > 0 || b > 0)
!Logical NOT (reverses condition)if (!condition)
&Boolean logical AND (evaluates both operands)if (a > 0 & b > 0)
|Boolean logical OR (evaluates both operands)if (a > 0 | b > 0)
^Exclusive OR (one true, one false)if (a > 0 ^ b > 0)
  • Precedence: ! > && > ||.
