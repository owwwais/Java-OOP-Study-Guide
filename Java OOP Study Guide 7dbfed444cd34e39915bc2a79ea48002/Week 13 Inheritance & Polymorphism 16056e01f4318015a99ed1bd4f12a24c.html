Inheritance & Polymorphism

1. Superclasses and Subclasses

  • Inheritance: Mechanism where one class (subclass) acquires properties and behaviors of another class (superclass).
  • Key Features:
    • Promotes software reusability.
    • Establishes "IS-A" relationships (e.g., a car is a vehicle).
    • A subclass can customize and add new fields or methods.
  • Types of Inheritance:
    1. Single Inheritance: One subclass inherits from one superclass.
    1. Multilevel Inheritance: A subclass inherits from another subclass.
    1. Hierarchical Inheritance: Multiple subclasses inherit from one superclass.
  • Examples:
    class Animal { void eat() { System.out.println("Eating..."); } }
	class Dog extends Animal { void bark() { System.out.println("Barking..."); } }

2. Protected Members

  • Definition: Allows access to class members within the same package and subclasses.
  • Syntax:
    protected int height;
	protected void setHeight(int h) { this.height = h; }

3. Relationship Between Superclasses and Subclasses

  • IS-A Relationship: A subclass object can be treated as a superclass object.
    • Example: A CarLoan is a Loan.
  • Super Keyword:
    • Refers to the superclass's members.
    • Example:
      class Subclass extends Superclass {
		void display() {
			super.display(); // Calls superclass method
		}
	}

4. Polymorphism

  • Definition: Ability of an object to take many forms, allowing a single action to be performed differently.
  • Types:
    1. Method Overloading: Multiple methods with the same name but different parameter lists.
    1. Method Overriding: Subclass redefines a method in the superclass.
  • Example:
    class Shape { void draw() { System.out.println("Drawing Shape"); } }
	class Circle extends Shape { void draw() { System.out.println("Drawing Circle"); } }

5. Abstract Classes and Methods

  • Abstract Class:
    • Cannot be instantiated.
    • Example:
      abstract class Animal {
		abstract void sound();
	}
  • Abstract Method:
    • Declared without implementation; must be defined in a subclass.
    • Example:
      class Dog extends Animal {
		void sound() { System.out.println("Bark"); }
	}
  • Interfaces:
    • Completely abstract classes to group related methods.
    • Example:
      interface Animal {
		void sound();
		void sleep();
	}
